# Xinerxin Trading Co., Ltd. - Corporate Website

## 项目概述 (Project Overview)

这是一个专业的欧美风格企业官网，专为深圳市鑫润欣贸易有限公司（Shenzhen Xinerxin Trading Co., Ltd.）设计开发。

This is a professional European-style corporate website developed for Shenzhen Xinerxin Trading Co., Ltd., specializing in mobile app management solutions and global trading services.

## 核心业务 (Core Services)

- **移动应用管理 (Mobile App Management)**: 应用发布、优化、变现服务
- **供应链解决方案 (Supply Chain Solutions)**: 采购、物流、库存管理
- **产品分销 (Product Distribution)**: 电子产品、日用品、农产品分销
- **进出口服务 (Import/Export Services)**: 国际贸易、报关、文件服务
- **商业咨询 (Business Consulting)**: 市场进入、扩张、运营优化

## 网站结构 (Website Structure)

```
/
├── index.html          # 首页 (Homepage)
├── services.html       # 业务服务 (Services)
├── culture.html        # 企业文化 (Corporate Culture)
├── news.html           # 企业动态 (News)
├── contact.html        # 联系我们 (Contact Us)
├── privacy.html        # 隐私政策 (Privacy Policy)
├── terms.html          # 服务条款 (Terms of Service)
├── robots.txt          # 搜索引擎指令
├── sitemap.xml         # 网站地图
├── app-ads.txt         # 移动广告验证
├── browserconfig.xml   # Windows磁贴配置
├── css/
│   └── style.css       # 主样式文件
└── js/
    └── main.js         # 主交互脚本
```

## 技术特性 (Technical Features)

- **响应式设计 (Responsive Design)**: 完美适配桌面端、平板、移动端
- **现代化UI/UX**: 专业的蓝白配色，流畅的动画效果
- **SEO优化 (SEO Optimized)**: 完整的meta标签、sitemap、robots.txt
- **无障碍访问 (Accessibility)**: 支持键盘导航、屏幕阅读器
- **跨平台兼容 (Cross-platform)**: 兼容所有现代浏览器
- **GDPR合规 (GDPR Compliant)**: 符合欧盟通用数据保护条例

## 部署说明 (Deployment Instructions)

1. 将所有文件上传到Web服务器根目录
2. 确保服务器支持HTTPS（推荐）
3. 配置域名解析到服务器
4. 验证sitemap.xml和robots.txt可访问
5. 提交sitemap.xml到Google Search Console
6. 配置SSL证书启用HTTPS
